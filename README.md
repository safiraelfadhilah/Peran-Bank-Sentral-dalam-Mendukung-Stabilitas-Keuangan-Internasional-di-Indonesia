# Peran-Bank-Sentral-dalam-Mendukung-Stabilitas-Keuangan-Internasional-di-Indonesia
Stabilitas keuangan internasional merupakan prasyarat penting bagi keberlanjutan pertumbuhan ekonomi suatu negara, terutama di tengah meningkatnya integrasi pasar keuangan global. Sebagai negara yang terbuka terhadap arus modal internasional, Indonesia menghadapi berbagai risiko eksternal.
